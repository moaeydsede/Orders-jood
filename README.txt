خطوات الرفع (GitHub Pages) — VIP:
1) فك الضغط.
2) ارفع الملفات (index.html + styles.css + app.js) في ROOT داخل الريبو (ليس داخل مجلد).
3) GitHub → Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main / (root)
4) افتح الرابط مع كسر الكاش:
   https://YOURNAME.github.io/REPO/?v=1

مهم جدًا لتسجيل الدخول:
- شيت "المستخدمين" يجب أن يكون:
  A1: اسم المستخدم
  B1: كلمة المرور
  (ثم بيانات من الصف 2)
- مشاركة الملف: أي شخص لديه الرابط = Viewer
- File → Share + File → Publish to web
